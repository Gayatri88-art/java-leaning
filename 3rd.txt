##Operators
**Arithmetic Operators: + (Addition), - (Subtraction), * (Multiplication), / (Division), % (Modulus)

**Unary Operators: + (Unary plus), - (Unary minus), a++ (Post-increment), ++a (Pre-increment), a-- (Post-decrement), --a (Pre-decrement)

**Relational Operators: == (Equal to), != (Not equal to), > (Greater than), < (Less than), >= (Greater or equal), <= (Less or equal)

**Logical Operators: && (AND), || (OR), ! (NOT)

**Short-circuit Logical Operators: Same as logical (&&, ||), but second part executes only if needed




### **Assignment Operators in Programming**

1. **Simple Assignment (`=`)**

   * Assigns the value on the right to the variable on the left.
   * **Syntax:** `a = 5;`

2. **Addition Assignment (`+=`)**

   * Adds the right-hand value to the variable and assigns the result to the variable.
   * **Syntax:** `a += 5;`
   * Equivalent to: `a = a + 5;`

3. **Subtraction Assignment (`-=`)**

   * Subtracts the right-hand value from the variable and assigns the result to the variable.
   * **Syntax:** `a -= 5;`
   * Equivalent to: `a = a - 5;`

4. **Multiplication Assignment (`*=`)**

   * Multiplies the variable by the right-hand value and assigns the result to the variable.
   * **Syntax:** `a *= 5;`
   * Equivalent to: `a = a * 5;`

5. **Division Assignment (`/=`)**

   * Divides the variable by the right-hand value and assigns the result to the variable.
   * **Syntax:** `a /= 5;`
   * Equivalent to: `a = a / 5;`

6. **Modulus Assignment (`%=`)**

   * Assigns the remainder of the division of the variable by the right-hand value.
   * **Syntax:** `a %= 5;`
   * Equivalent to: `a = a % 5;`
